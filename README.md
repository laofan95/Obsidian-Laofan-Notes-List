# Laofan Notes List

Obsidian 插件，显示纯笔记列表，按最新修改时间排序，类似微信收藏样式。

## 功能特性

### 📋 核心功能
- 按最新修改时间排序的笔记列表
- 类似微信收藏的简洁界面设计
- 支持无限滚动加载更多笔记
- 实时更新，自动响应笔记变化

### 🔍 筛选与搜索
- **分类筛选**：支持按「最近使用」、「笔记」、「链接」、「图片」、「视频」、「图片和视频」、「文件」分类查看
- **标签筛选**：支持按笔记标签筛选
- **全局搜索**：可搜索笔记标题、内容、标签等
- **组合筛选**：支持分类+标签+搜索的组合筛选

### 🎨 个性化设置
- 自定义每页显示的笔记数量
- 自动滚动到当前打开的笔记
- 显示/隐藏笔记缩略图
- 设置每个笔记最多显示的标签数量
- 启用/禁用删除附件提示
- 启用/禁用删除文件夹提示
- 显示/隐藏时间提示
- 始终聚焦当前标签页

### 📁 智能管理
- 支持删除笔记时选择是否删除附件
- 支持删除文件夹时选择「只删除文件夹」或「删除全部笔记/附件」
- 鼠标悬停显示创建时间和最后修改时间
- 支持快速创建新笔记
- 支持快速返回顶部

## 安装方法

### 方法一：从 Obsidian 社区插件市场安装
1. 打开 Obsidian 设置
2. 点击「第三方插件」
3. 搜索「Laofan Notes List」
4. 点击「安装」按钮
5. 安装完成后点击「启用」按钮

### 方法二：手动安装
1. 下载插件的最新版本 zip 文件
2. 解压到 Obsidian 库的 `.obsidian/plugins/` 目录下
3. 重启 Obsidian
4. 在设置的「第三方插件」中启用插件

## 使用说明

### 打开笔记列表
1. 在 Obsidian 左侧边栏点击「笔记列表」图标
2. 或使用快捷键打开（可在 Obsidian 设置中自定义）

### 浏览笔记
- 笔记列表按最新修改时间排序，最新的笔记在最上方
- 向下滚动可加载更多笔记
- 点击笔记标题可打开笔记
- 点击「回到顶部」按钮可快速返回列表顶部

### 筛选笔记
1. **按分类筛选**：点击顶部分类标签（最近使用、笔记、链接等）
2. **按标签筛选**：点击下拉按钮，选择标签分类
3. **搜索笔记**：在搜索框中输入关键词
4. **清除筛选**：点击筛选标签上的「×」按钮

### 创建新笔记
点击右上角的「+」按钮可快速创建新笔记

### 删除笔记
1. 右键点击笔记项
2. 选择「删除」选项
3. 根据提示选择是否删除附件

### 删除文件夹
1. 右键点击文件夹
2. 选择「删除」选项
3. 根据提示选择「只删除文件夹」或「删除全部笔记/附件」

## 设置选项

### 基本设置
- **每页显示的笔记数量**：设置每次加载显示的笔记数量（默认：20）
- **自动滚动到当前笔记**：打开笔记列表时，自动滚动到当前打开的笔记位置

### 显示设置
- **显示缩略图**：在笔记列表中显示图片和文件的缩略图
- **每个笔记最多显示的标签数量**：设置每个笔记项中最多显示的标签数量（默认：3）
- **显示时间提示**：鼠标悬停在笔记项上时，显示创建时间和最后修改时间的提示

### 安全设置
- **启用删除附件提示**：删除包含附件的笔记时，显示确认对话框询问是否同时删除附件
- **启用删除文件夹提示**：删除包含笔记的文件夹时，显示确认对话框询问删除方式

### 标签页设置
- **始终聚焦当前标签页**：启用后，笔记会在同一个标签页打开；如果标签页被关闭，下次点击会在新标签页打开。Ctrl+点击可强制在新标签页打开。
